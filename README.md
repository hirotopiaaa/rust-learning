- [The Rust Programming Language](https://doc.rust-lang.org/book/)

## Initialize a project

```bash
cargo new <project-name>
```

## Build a project

```bash
cargo build
```

The executable binary is located in `./target/debug/<project-name>`.

## Run a project

```bash
cargo run
```

## Check a project

```bash
cargo check
```
